# People Search Benchmark

An open benchmark for evaluating AI-powered people search platforms across influencer discovery, recruitment, and lead generation.

Built by [Lessie AI](https://lessie.ai).

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Push to GitHub and import on [Vercel](https://vercel.com/new). No additional configuration needed.
